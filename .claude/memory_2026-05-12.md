## Task 3: 骑车人跟踪 — 当前方案

### 四锚点样条轨迹
```
Frame   0: (253, 179)
Frame  50: (408, 208)
Frame 100: (448, 320)
Frame 140: (528, 488)
Frame 142: 骑手消失，停止跟踪
```

### 核心挑战
NCC跨帧相似度极低（帧0→帧30: 0.00，帧0→帧10: -0.15）。骑手目标小（~25px），外观在不同帧间剧烈变化，单模板无法覆盖全程。

### 最终方案: 滑动模板 NCC + 样条约束 + 卡尔曼
- **模板**: 30×44 px (高>宽匹配骑手身形)，从置信NCC帧提取并更新
- **NCC需求**: ncc≥0.30 AND 距K预测≤12px AND 距样条≤24px 三者同时满足
- **失败回退**: 直接使用样条位置，卡尔曼状态重置
- **NCC命中率**: ~31.7%
